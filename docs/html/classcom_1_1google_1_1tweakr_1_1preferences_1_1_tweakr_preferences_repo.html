Creates a local repo that saves values to the specified local shared preferences.

Parameters
context
valuesPreferencesThe SharedPreferences to store values in. This must be the same as your PreferenceFragment uses.
metadataPrefsThe SharedPreferences to store internal metadata about Tweak fields. Best practice is to use a unique file name so that it doesn't pollute your other preferences (i.e. this should be different than valuesPreferences).
